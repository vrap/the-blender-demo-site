The philosophy

The project itself was designed to be modular, with openhardware and software let anyone improving the system and making the machine. This possibility let you:

  • Clone The blender with a lasercutter, 3D printer and some electronic materials.
  • Fork The blender, to improve it and propose new feature for the community.
  • Create or join a community of cocktail lover and share your creations with your friends.
  • Expand your Blender with new module (also printable and lasercuttable).

The simplicity aspect of the project has been in our mind throughout the duration of the project and this is why we have documented all parts in this wiki. Here you can found some informations about:

  • Materials requirements
  • Hardware plans
  • Software explanations
  • And a lot more stuff that we think it's necessary for you

If you think there is a hole about some informations you can improve it by making a pull request or let us know by adding an issue to the documentation repository.

And last but not least, the lowcost aspect was a constraint of our project since we have not a big budget, think that existing cocktail dispenser are too expensive at this time and because we think that it's more fun to let anyone to make his Blender by themselves.
