About the project

Modularity

The project itself was designed to be modular and with openhardware and software let anyone improving the system and making the machine.

Simplicity

The simplicity aspect of the project has been in our mind throughout the duration of the project and this is why we have documented all aspect of the project. Learn more with the documentation

Lowcost

We think that existing cocktail dispenser are too expensive at this time and it's more fun to let anyone to make his Blender by themselves.
