Presentation

The Blender is an automatic and connected cocktail dispenser, which was created with the following principles:

  • open-source
  • modularity
  • lowcost
This cocktail dispenser is a project of four students from the University of Cergy-Pontoise (France), in the formation licence professionnelle d√©veloppeur & Web mobile. It is realized in the context of specialization course about connected objects and Internet of things.

As mentionned above, the project was designed to be open-source, and let you:
  • To clone The Blender with help of some tools, as lasercutter, 3D printer and some electronic materials.
  • To fork The Blender, and improve it or propose new feature for others users.
  • To create or join a community of cocktails lover and share your creations with your friends.
  • To expand your Blender with new module.
The simplicity aspect of the project has been in our mind throughout the duration of the project and you can found some informations about:
  • materials requirements
  • hardware plans
  • software explanations
  • a lot more stuff that we think it's necessary for you
If you think there is a hole about some informations, you can improve it by making a pull request or let us know by adding an issue to the documentation repository.

Hope you enjoy this project, feel free to comment and improve it.
