Getting started

La documentation suivante est crée pour vous permettre de fabriquer vos propres modules et d'assembler The Blender.
Nous avons essayés de rendre ces explications les plus claires possibles, mais les principes, techniques et technologies utilisés ne sont pas forcément simples. Si vous trouvez que la documentation peut être améliorée, n'hésitez pas à nous en faire part.
Enjoy!

Installation

Il y a 2 parties que vous pouvez installer :

  • l'application
  • la communauté

Application

Pré-requis

MEAN est un acronyme pour MongoDB, Express, AngularJs et NodeJs. The Blender utilise cette architecture.

Node.js

Téléchargez et installez Node.js

MongoDB

Téléchargez et installez MongoDB. Assurez vous qu'il tourne sur le port par défaut (27017).

NPM

Gestionnaire de paquets de Node.js.

Installation

npm install the-blender

Options

L'application a besoin d'être configurée en fonction de votre propre environnement. Nous essayé de la réduire au minimum, afin que cette étape reste simple.

Ci-dessous le fichier de configuration générique de l'application. Nous recommandons de ne pas le modifier, mais d'en faire une copie et de modifier celle-ci.

// Copie du fichier original vers un fichier personnalisé
cp config.sample.js config.js
            
/**
 * config.js
 */
module.exports = {
    database: {
        url: 'mongodb://localhost:25486/the-blender'
    },
    master: {
        url: 'http://localhost:8080/'
    },
    server: {
        port: 5555
    },
    board: {
        port: '/dev/ttyACM0',
        debug: true,
        cart: {
            A: {
                pwm: 3,
                brake: 9,
                dir: 12
            },
            B: {
                pwm: 11,
                brake: 8,
                dir: 13
            }
        }
    }
};
